## Summary

| Параметр | Значение |
|---|---|
| Тип задачи | Historical planning |
| Вход | TXT-бриф: план на август, бюджет 22 289 884 ₽ без НДС, площадки подобрать по бенчам и рекомендациям прошедших РК. |
| Выход | Excel: МП + ГЕО. Client10, 01.08–31.08, объём ЦА 62 млн, Video/Display, MTS Multi-roll/in-stream, CPM, гео-регионы. |

## Planner assumptions

- MTS/video/display.

- Гео-регионы.
- Площадочный сплит из истории.

## MVP rules

- “по бенчам/прошлым РК” → искать historical reference.
- Нет истории → blocker/warning.
- Короткие брифы → список допущений.
- Режим бюджета хранить явно.
