# Сводная матрица правил

| Признак брифа | Класс кампании | Шаблон/вкладки | Основные расчёты | Риски |
|---|---|---|---|---|
| HR, вакансия, VK, малый бюджет | HR LeadAds | VK/соцсети | CPC, клики, лиды, CPL | нет гео/периода |
| Объекты + бюджеты | Multi-object context | Контекст + Итого | CPC, бюджет по объектам | дубли объектов, неверный итог |
| CPL, заявка, лендинг, форма | Performance | МП + семантика + минус-фразы | CR, лиды, CPL | нет посадочной/CR |
| OLV, баннеры, охват | Медийная | Digital media plan | CPM, охват, частота, VTR | нет бюджета/частоты/гео |
| Блогеры, TG/YT/VK | Influencer | Подборка блогеров | цена размещения, охват, ER | нет базы блогеров/цен |
| B2B/B2C, банковский продукт | Search + social | MediaPlan + семантика | CPC, клики, CPL | договорные цены, legal |
| Тендер, viewability/fraud/brand safety | Brand safety media | Digital + селлеры + quality | CPM, SOV, viewability | нужны данные площадок |
| “по бенчам/прошлым РК” | Historical planning | прошлый шаблон клиента | бенч CPM/CPC/CTR/CR | без истории система слепая |
| PPTX с несколькими блоками | Multi-campaign parsing | выбранный блок кампании | зависит от блока | нужно выбрать кампанию |

## Hard-stop проверки

- нет бюджета;
- нет периода;
- неясный режим бюджета: с НДС/без НДС/с АК;
- нет гео для геозависимых площадок;
- нет целевого действия для CPL/performance;
- requested “по бенчам”, но бенчи/история не загружены.

## Warning проверки

- нет KPI;
- нет посадочной;
- нет фактических данных прошлых РК;
- неясная ЦА;
- площадка указана, формат не указан;
- есть дубли объектов;
- бюджет слишком мал для выбранного числа площадок.
