# media-rules-mono

GitHub Pages-документация по правилам сопоставления клиентского брифа и digital-медиаплана.

Сайт: https://svoronkov-les13.github.io/media-rules-mono/

Содержит разбор всех исторических пар `Бриф_clientN → МП_clientN` и правила для MVP: бриф → структурированные вводные → сценарий → шаблон → площадки/форматы → сплит бюджета → Excel-медиаплан.
