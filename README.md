# media-rules-mono

GitHub Pages-документация по правилам сопоставления клиентского брифа и digital-медиаплана.

Сайт: https://svoronkov-les13.github.io/media-rules-mono/

Документ фиксирует минимальную логику MVP: бриф → структурированные вводные → сценарий → шаблон → площадки/форматы → сплит бюджета → Excel-медиаплан.
