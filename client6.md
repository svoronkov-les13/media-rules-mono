# Client6 — готовая еда, тендерная медийная кампания

[← На главную](index.html) · [Сводная матрица](matrix.html)

## 1. Что было в брифе

Маркетинговый бриф по готовой еде. Цели: рост чеков/РТО, новые клиенты, знание продукта. Медийные KPI: широкий охват light TV viewers, видимость, fraud/brand safety, расхождение с Weborama. Период июль–октябрь 2025. Бюджет Total media 6 млн ₽. Deliverables: OLV и banners.

## 2. Что получилось в медиаплане

Сложный digital-план: `МП Digital основной`, `Гео`, `Распределение по аудиториям`, отдельные листы Ozon, VK, Telegram Ads, Яндекс, WB, Sberseller, GPM. Есть баннеры, social/video/display, распределение по аудиториям и гео, BLS, тендерные условия, показатели охвата/частоты/CTR/CPM.

## 3. Сопоставление БРИФ → МЕДИАПЛАН

| Из брифа | В медиаплан |
|---|---|
| OLV + banners | становятся группами video/display/social/retail media |
| Гео зависит от креатива | вынесено на отдельную вкладку “Гео” |
| Несколько ЦА | разложены в “Распределение по аудиториям” |
| Тендерные KPI | должны стать ограничениями качества инвентаря |

## 4. Что медиапланер добавил сам

- Добавлены retail media селлеры Ozon/WB/Sberseller/GPM.
- Бюджет распределён между аудиториями и площадками по экспертной логике.
- BLS отмечается как опция/условие.
- Нужны справочники viewability/fraud/brand safety по площадкам.

## 5. Правила, которые надо вынести в MVP

- Если бриф содержит viewability/fraud/brand safety/Weborama — включать quality constraints в модель планирования.
- Для FMCG/retail/готовой еды с OLV+banners выбирать retail media + video/display микс.
- Гео/креативные ротации хранить отдельной вкладкой/таблицей.
- Для тендеров обязательно показывать обоснование площадок и соответствие KPI.

## 6. Что это значит для автоматизации

Для этой пары система должна уметь:

- классифицировать тип кампании;
- выбрать подходящий Excel-шаблон;
- перенести явные вводные из брифа в структуру медиаплана;
- отделить факты от экспертных допущений;
- подсветить недостающие данные;
- сгенерировать черновик, который медиапланер может быстро проверить и поправить.
