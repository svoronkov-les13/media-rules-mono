# Client2 — HR-вакансия кассира в VK

<div class="page-meta">Pair: <code>Бриф_client2</code> → <code>МП_client2</code></div>

## Summary

| Параметр | Значение |
|---|---|
| Тип задачи | HR / Lead Ads |
| Вход | TXT-бриф: разместить вакансию кассира в VK, бюджет до 50 000 ₽. |
| Выход | Excel: один лист VK-Реклама, одна строка VK LeadAds, CPC, бюджет с НДС, клики, прогноз лидов. |
| Ключевое правило | Малый HR-бюджет не дробится: один канал, один формат, быстрый расчёт отклика. |
| Главный риск | В брифе нет гео и периода — это warning/default, а не факт. |

## Brief fields

| Поле | Значение | Источник | Действие в системе |
|---|---|---|---|
| Кампания | Вакансия кассира | Brief | Объект рекламирования |
| Канал | VK | Brief | Единственная площадка |
| Бюджет | до 50 000 ₽ | Brief | Ограничение строки |
| Гео | Москва/МО | Planner assumption | Warning |
| Период | Октябрь | Planner assumption | Warning |

## Media plan structure

| Лист / блок | Назначение |
|---|---|
| ВК-Реклама | Строка LeadAds + итоговые формулы |

## Mapping rules

| Из брифа | В медиаплан |
|---|---|
| HR/вакансия | класс HR Lead generation |
| VK | выбор VK LeadAds |
| Бюджет 50k | стоимость с НДС |
| Формат не задан | LeadAds выбран по типу задачи |

## Planner assumptions

- Москва/МО.
- Октябрь.
- CTR/CPC/прогноз лидов из бенчей.

## MVP rules

- HR + VK + бюджет до 100k → одна строка VK LeadAds.
- Малый бюджет не дробить.
- Отсутствующие гео/период подсвечивать.
- Главный KPI: лиды/CPL, не охват.
