# Client1 — детские книги: блогеры + digital

<div class="page-meta">Pair: <code>Бриф_client1</code> → <code>МП_client1</code></div>

## Summary

| Параметр | Значение |
|---|---|
| Тип задачи | Influencer + digital support |
| Вход | DOCX-бриф: издательство детской литературы, женщины 25–45 с детьми 2–6, ЮФО + Москва/МО, две бюджетные опции 2 млн и 1,4 млн, блогеры TG/YT/VK. |
| Выход | Excel: свод, Digital, техтребования VK/Яндекс/Avito, блогеры-рекомендации и подборка. |
| Ключевое правило | Блогерский бриф превращается в гибридный план: influencer-блок отдельно, digital-поддержка отдельно. |
| Главный риск | Нужна база блогеров с ценами, тематикой, подписчиками и форматами. |

## Brief fields

| Поле | Значение | Источник | Действие в системе |
|---|---|---|---|
| Продукт | Детские развивающие книги | Brief | Шапка кампании и контекст подбора |
| ЦА | Женщины 25–45, дети 2–6 | Brief | Таргетинги Яндекс/VK/Avito и блогеры |
| Гео | ЮФО + Москва/МО | Brief | Геотаргетинг строк |
| Бюджет | 2 млн и 1,4 млн | Brief | Два сценария медиаплана |
| Форматы | Фото/видео посты | Brief | Варианты в influencer-блоке |

## Media plan structure

| Лист / блок | Назначение |
|---|---|
| свод | Итоговая агрегация площадок, форматов и бюджета |
| Digital | Яндекс, VK, Avito, прогноз кликов/показов/стоимости |
| ТТ VK/Яндекс/Avito | Технические и юридические требования |
| Блогеры реко/подборка | Обязательные и рекомендованные блогеры |

## Mapping rules

| Из брифа | В медиаплан |
|---|---|
| Список желаемых блогеров | попадает в отдельную вкладку, не смешивается с performance-строками |
| Родительская тематика | превращается в parenting/kids-сегменты |
| Две бюджетные опции | генерируют две версии/сценария |
| Формат не определён | система предлагает фото и видео как альтернативы |

## Planner assumptions

- Период “сентябрь” добавлен экспертно.
- Digital-площадки добавлены сверх блогерского запроса.
- Техтребования добавлены из справочников площадок.

## MVP rules

- Если есть блогеры — создавать отдельный influencer-блок.
- Обязательных блогеров показывать даже при высокой цене.
- Для детской/родительской ЦА использовать VK/Яндекс/Avito + parenting-блогеров.
- Несколько бюджетов = несколько сценариев.
